## 2. 단축키

[공식문서](https://code.visualstudio.com/shortcuts/keyboard-shortcuts-windows.pdf)

* 검색
    * 전체폴더에서 검색
        * ctrl + shift + f
    * 파일이름을 검색
        * ctrl + shift + e
    * Replace
        * ctrl + h
* 라인단위 처리
    * 코드라인을 Move
        * Alt + up/down
    * 코드라인을 Copy
        * Alt + Shift + up/down
* Insert Cursor : 여러곳에 커서만들기
    * alt + click
* 해당단어를 파일에서 전체선택
    * 단어를 블록선택후 ctrl + D
